## Output
The output file is located in each libraries output folder it will contain output file for all libraries used (TextBlob, NLTK, spaCy) which include

    *_cleaned.txt - The cleaned text
    *_words.txt - Tokenized words
    *_top10words.txt - Top 10 most frequent words
    timecompared.txt - A table comparing the runtime of each library (NLTK, spaCy, TextBlob)
